# scoop-go-playball

Scoop bucket for [go-playball](https://github.com/pdavlin/go-playball) — a terminal-based MLB game viewer.

## Install

```powershell
scoop bucket add go-playball https://github.com/pdavlin/scoop-go-playball
scoop install go-playball
```

## Update

```powershell
scoop update go-playball
```

This manifest is automatically updated by CI when a new release is published.
